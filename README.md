# Zancord Manager

![Open Software License 3](https://img.shields.io/badge/License-OSL%20v3-yellow.svg)

Application to use Zancord on Android.

## About

Zancord Manager is an application that installs and updates [Zancord](https://github.com/zanfiel/zancord-mobile),
a modification for Discord.

## Features

Some of the features Zancord Manager provides are:

- **Install**: Download Discord and install Zancord on it
- **Update**: Update Zancord to the latest version
- **Notifications**: Be notified when a new version of Zancord is available
- **Changelog**: See the changelog of Zancord
